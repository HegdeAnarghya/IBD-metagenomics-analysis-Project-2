## Overview

This project performs whole genome shotgun metagenomic analysis of IBD patients using pre-computed profile data from the HMP2 (iHMP) study. Unlike 16S rRNA sequencing (Project 1), metagenomics provides:

- **Species-level taxonomic resolution** (not just genus)
- **Direct functional measurement** (genes and pathways quantified)
- **EC enzyme-level precision** (1,537 classified enzymes)
- **Species–function correlations** (which bacteria drive which pathways)

**Research Questions:**
1. How do gut microbiomes differ between healthy and IBD patients at species level?
2. Which metabolic pathways are disrupted in IBD?
3. Which bacterial species correlate with functional pathway changes?
4. How does metagenomics compare to 16S rRNA sequencing?

**Companion to:** [Project 1 — 16S rRNA Analysis](https://github.com/HegdeAnarghya/ibd-microbiome-analysis)

## Methods

### Important Note on Data Processing
This project uses **pre-computed profile files** provided by the HMP2 consortium — MetaPhlAn taxonomic profiles and HUMAnN functional profiles were generated by the HMP2 team, not in this analysis. Our analysis begins from those output files.

### Our Analysis Pipeline

**Session 1 — Taxonomic Profiling (`01_explore_taxonomic_profiles.py`)**
1. Load MetaPhlAn species abundance profiles
2. Calculate alpha diversity (Shannon index, species richness)
3. Build species abundance matrix (109 × 5)
4. Visualize diversity and top species per sample

**Session 2 — Functional Pathway Analysis (`02_functional_pathways.py`)**
1. Load HUMAnN pathway abundance files
2. Filter to unstratified pathways
3. Build pathway abundance matrix (312 × 5)
4. Visualize top 15 pathways as heatmap

**Session 3 — 16S vs WGS Comparison (`03_compare_16S_vs_metagenomics.py`)**
1. Compare resolution capabilities (genus vs species)
2. Contrast functional data availability
3. Compare diversity metrics between approaches
4. Visualize complementary strengths

**Session 4 — Strain, Gene Family & Enzyme Analysis (`04`, `05`, `06`)**
1. Strain-level variation tracking across samples
2. UniRef90 gene family abundance profiling
3. EC enzyme classification and annotation
4. Identification of key enzymes (butyrate kinase, beta-lactamase)

**Session 4b — Differential Analysis (`07_differential_analysis.py`)**
1. Compare Healthy vs IBD (excluding HSM6XRQE outlier where noted)
2. Wilcoxon rank-sum test (non-parametric, appropriate for n=5)
3. Calculate fold changes
4. Generate volcano plots for pathways and enzymes

**Session 5 — Species–Function Correlation (`08_species_function_correlation.py`)**
1. Spearman correlations: species abundance vs pathway abundance
2. Spearman correlations: species abundance vs EC enzyme abundance
3. Species co-abundance heatmap
4. Identify top positive and negative associations

**Session 6 — Machine Learning (`09_machine_learning.py`)**
1. Random Forest classifier — IBD vs nonIBD
2. Leave-One-Out Cross-Validation (LOOCV, appropriate for n=5)
3. Feature importances for top predictive pathways
4. ROC curve vs 16S baseline (AUC = 0.894)

**Session 7 — Visualization Dashboard (`10_visualization_dashboard.py`)**
1. 6-panel comprehensive summary figure
2. Alpha diversity, species abundance, pathway heatmap
3. ML feature importances, species–pathway correlations
4. Project summary statistics table

### Statistical Methods
- **Alpha diversity:** Shannon index H = −Σ(pᵢ × ln(pᵢ))
- **Differential analysis:** Wilcoxon rank-sum test (non-parametric)
- **Correlations:** Spearman rank correlation
- **ML validation:** Leave-One-Out Cross-Validation
- **Note:** With n=5, all statistical results are exploratory; no multiple testing correction applied

### Software
- Python 3.10
- pandas (data manipulation)
- numpy (numerical operations)
- scipy (Wilcoxon tests, Spearman correlations)
- scikit-learn (Random Forest, LOOCV)
- matplotlib, seaborn (visualization)

## Results

### 1. Species-Level Alpha Diversity

**Shannon Diversity Index:**
- Healthy (HSM7CZ2A): 2.43
- CD (CSM5FZ4C): 1.87 (23% lower than healthy)
- UC range: 1.17–2.65 (2.3-fold variation)
- HSM6XRQE: 1.17 (52% lower than healthy)

**Species Richness:**
- Healthy: 56 species
- CD: 28 species (50% reduction)
- UC range: 9–63 species (7-fold variation)
- HSM6XRQE: 9 species (84% reduction)

**Key Observation:** Extreme heterogeneity within the UC diagnosis. Two UC patients (MSM5LLHV, CSM9X1ZO) show diversity comparable to or exceeding the healthy control, while HSM6XRQE shows near-complete collapse.

### 2. Functional Pathway Analysis

**Total unique pathways detected:** 312

**Pathway counts per sample:**
- MSM5LLHV (UC): 296 pathways
- HSM7CZ2A (Healthy): 233 pathways
- CSM9X1ZO (UC): 230 pathways
- CSM5FZ4C (CD): 209 pathways
- HSM6XRQE (UC): 9 pathways

**Top 5 pathways by mean abundance:**
1. DTDPRHAMSYN-PWY: dTDP-L-rhamnose biosynthesis I
2. PWY-7219: Adenosine ribonucleotides de novo biosynthesis
3. PWY-5695: Urate biosynthesis / inosine 5'-phosphate degradation
4. ILEUSYN-PWY: L-isoleucine biosynthesis I
5. VALSYN-PWY: L-valine biosynthesis

### 3. EC Enzyme Analysis

**Total unique EC enzymes:** 1,537

**Total enzymatic capacity (CPM):**
- HSM7CZ2A (Healthy): 848,097 CPM (100%)
- CSM9X1ZO (UC): 819,060 CPM (97%)
- CSM5FZ4C (CD): 665,928 CPM (79%)
- MSM5LLHV (UC): 482,174 CPM (57%)
- HSM6XRQE (UC): 45,327 CPM (5%)

### 4. Differential Analysis (Healthy vs IBD)

- Wilcoxon rank-sum tests applied; results exploratory at n=5
- **All 20 pathways analyzed were depleted in IBD** — none were enriched
- Top differentially depleted: Glycolysis IV, L-lysine biosynthesis III, folate transformations, peptidoglycan biosynthesis
- Butyrate kinase (EC 2.7.2.7): 125 CPM (healthy) vs 85 CPM (IBD avg) — 32% reduction
- Beta-lactamase (EC 3.5.2.6): 2.55× higher in healthy (expected — normal gut microbiome carries resistance genes)

### 5. Species–Function Correlations

**Spearman correlation highlights:**
- ***Bacteroides vulgatus*** — strongest positive correlations with IBD-associated metabolic pathways
- ***Faecalibacterium prausnitzii*** — negative correlations with IBD-elevated pathways (consistent with its anti-inflammatory role and known depletion in IBD)
- Co-abundance heatmap revealed two primary microbial guilds:
  1. *Bacteroides*-dominated guild (positively correlated with IBD pathways)
  2. *Faecalibacterium*-dominated guild (negatively correlated)

### 6. Machine Learning (LOOCV, n=5)

| Metric | Value |
|--------|-------|
| Algorithm | Random Forest |
| Validation | Leave-One-Out CV |
| AUC (WGS, n=5) | 0.000 |
| AUC (16S baseline, n=1,233) | 0.894 |

**Top feature importances:**
1. tRNA charging pathway (TRNA-CHARGING-PWY)
2. L-lysine biosynthesis III (PWY-2942)
3. Peptidoglycan biosynthesis (PEPTIDOGLYCANSYN-PWY)

**Interpretation:** AUC = 0.000 is an expected artifact of 4:1 class imbalance with LOOCV at n=5 — the single nonIBD sample receives the highest predicted IBD probability because the model trains on IBD-only data in that fold. This is a methodological demonstration, not a model failure.

## Biological Interpretation

### HSM6XRQE — Severe Dysbiosis Case

This UC sample stands as a striking case of near-complete microbiome collapse:
- Only 9 species detected (vs 28–63 in all other samples)
- Only 9 active metabolic pathways (vs 209–296)
- 5% of healthy total enzymatic capacity (45,327 vs 848,097 CPM)
- Zero butyrate synthesis capacity
- Single species (*Prevotella stercorea*) dominates at ~69%

This pattern is consistent with severe active inflammation, possibly compounded by recent antibiotic exposure. This patient would represent an entirely different clinical state from the other UC patients.

### UC Heterogeneity

The three UC patients span an enormous functional range:

| Metric | MSM5LLHV | CSM9X1ZO | HSM6XRQE |
|--------|----------|----------|----------|
| Species richness | 63 | 56 | 9 |
| Shannon | 2.65 | 2.65 | 1.17 |
| Pathways | 296 | 230 | 9 |
| Total CPM | 482,174 | 819,060 | 45,327 |

This demonstrates that "UC" encompasses vastly different microbiome states — a key argument for personalized, functional-capacity-based treatment approaches.

### Butyrate Synthesis
Butyrate is a short-chain fatty acid with anti-inflammatory properties that feeds intestinal epithelial cells and strengthens the gut barrier. Its depletion is a consistent finding in IBD.
- Healthy: 125 CPM butyrate kinase (EC 2.7.2.7)
- IBD average: 85 CPM (32% reduction)
- HSM6XRQE: 0 CPM (complete loss)

### Functional Redundancy
Multiple bacterial species can perform the same metabolic function (e.g., butyrate production). Metagenomics measures the **net functional capacity** — the sum across all species — which is more clinically relevant than tracking any single organism.

## Comparison to 16S

| Feature | 16S (Project 1) | Metagenomics (Project 2) |
|---------|----------------|--------------------------|
| **Samples** | 1,233 | 5 |
| **Resolution** | Genus-level | Species-level |
| **Features** | 8,513 OTUs | 109 species |
| **Functional data** | None | 312 pathways, 1,537 enzymes |
| **ML AUC** | 0.894 | 0.000 (n=5 limitation) |
| **Cost** | $50–100/sample | $200–500/sample |
| **Strength** | Statistical power | Functional precision |

**Key agreements between both methods:**
- IBD patients have lower diversity than healthy
- *Bacteroides* genus is abundant across groups
- Individual variation is high within IBD

**What metagenomics reveals beyond 16S:**
- Species-level identification within genera
- Direct functional capacity quantification
- Enzyme-level resolution (1,537 EC enzymes)
- Extreme individual variation (up to 850-fold within same diagnosis)

**Conclusion:** The two approaches are complementary — use 16S for large-scale discovery, metagenomics for mechanistic depth.

## Limitations

| Limitation | Impact | Notes |
|------------|--------|-------|
| n=5 samples | No statistical power; all results exploratory | Apply pipeline to full HMP2 cohort (n>100) for real inference |
| Pre-computed profiles | Cannot re-analyze with different parameters | Download raw FASTQ for full pipeline control |
| Class imbalance (4:1) | ML AUC = 0.000 | Expected artifact; need balanced classes and larger n |
| Cross-sectional design | Cannot track progression or treatment response | HMP2 has longitudinal samples available |
| No clinical metadata integration | Cannot correlate with disease activity scores | Full HMP2 metadata is downloadable |
| No validation cohort | Findings not independently replicated | Clearly a pilot/learning project |

## Conclusions

### Main Findings

1. **109 species** detected via MetaPhlAn taxonomic profiles — species-level precision beyond 16S
2. **1,537 EC enzymes** and **312 metabolic pathways** quantified directly from HUMAnN functional profiles
3. **Extreme individual variation** within UC: 7-fold range in species richness, 18-fold in total enzymatic capacity
4. **Butyrate synthesis depleted** in IBD: 32% average reduction; complete loss in one UC patient
5. ***B. vulgatus*** positively and ***F. prausnitzii*** negatively correlated with IBD-associated pathways — consistent with published literature
6. **All pathways depleted** in IBD (none enriched) — global metabolic suppression pattern
7. **ML at n=5 is underpowered** — AUC = 0.000 is a methodological lesson, not a failure

### Key Lessons

**1. Functional > Taxonomic:**
What bacteria *do* matters more than which bacteria are present. Metagenomics measures net functional capacity across all species.

**2. IBD is heterogeneous:**
UC is not one microbiome state. Functional capacity ranges from near-healthy to near-zero within the same diagnosis.

**3. Sample size determines ML performance:**
LOOCV at n=5 with 4:1 imbalance is incapable of meaningful classification. Contrast with 16S AUC=0.894 at n=1,233 illustrates the sample size requirement for microbiome ML.

**4. 16S + Metagenomics = Comprehensive:**
Project 1 provided discovery power (n=1,233). Project 2 provides mechanistic depth (n=5, full functional profiles). Both are necessary for complete understanding.

### Impact

**Scientific:**
- Demonstrates feasibility of WGS metagenomics analysis on a standard laptop (8GB RAM)
- Validates published HMP2 functional findings in independent Python analysis
- Provides fully reproducible open-source pipeline

**Educational:**
